Find Groove

1. Choose a folder(directory) that contains *.wav file(s).

2. Make a fingerprint file. If 'fingerprints' is already in the path, skip this process.

3. Verify that the tensorflow model is in the path. See 'https://github.com/Ground17/find_groove' for a detailed description of the model.

4. Everything is ready! Press the 'Recording' button below to start searching for the song!
